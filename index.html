windows下如何配置Apache2.4

下载Apache2.4

下载地址

解压下载的压缩包

解压后，压缩包里面有一个名为Apache24的目录，它就是我们需要的。

将它放置到你想要的路径下（例如我放在”D:/Apache24”）。

修改配置文件

配置文件路径：Apache24\conf\httpd.conf

修改项含义以及如何修改

  • Define SRVROOT “c:/Apache24”
    1. 含义：定义一个变量  Apache服务器目录
    2. 修改：我将它修改为了”D:/Apache24”
  • Listen 80
    1. 含义：Apache监听哪个端口
    2. 修改：可以不改，也可以改成一些常见的开发环境端口如：3000、5000、8000等
  • DocumentRoot “${SRVROOT}/htdocs”
    1. 含义：服务器网站根目录
    2. 修改：我将它修改为了DocumentRoot “F:/ROOT”（注：自己创建F:\ROOT这个目录）
    1. 含义：允许访问的目录（当然改成网站跟目录啦）
    2. 修改：

到这里服务器就已经可以使用了。下面以我自己配置的Apache服务器为例，讲述如何使用

打开Apache24\bin\httpd.exe这个应用程序即开启服务器

访问localhost:3000（你设置了哪个端口号就输入哪个端口号，默认80端口，即不用输入端口号）

告诉我，你成功了！

创建Apache服务

  • 打开cmd，进入Apache24\bin这个目录
  • 输入  httpd -k install -n “Apache”  安装Apache服务
  • 进入服务，可以看到Apache服务
  • 在cmd中输入  net start apache（此处服务名大小写均可）即可开启Apache服务
  • 在cmd中输入  net stop apache  即可关闭Apache服务
  • 输入  httpd -k uninstall -n “Apache”  卸载Apache服务
